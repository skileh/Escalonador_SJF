# Compilação:
Para compilar o programa digite no terminal:
> gcc -o nome_prog.o nome_prog.c

Exemplo:
> gcc -o exec.o Escalonador.c

Para execução do programa digite:
> ./nome_programa nome_arquivo qnt_processadores

Exemplo:
> ./exec.o tarefas_1.txt 3
